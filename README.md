# English Learning Strategy · 沪教版八年级上册 单词/短语 学习卡

一个面向初中生的英语单词学习网页应用：**267 条**词汇（单词 203 + 短语 64），自带**自然语音音频**、**中英对照例句**、**Anki 式卡片**与**听写错误计数**。

> 📖 教材：上海教育出版社《英语》八年级上册（Units 1–8）。教材词汇表以**英式发音**为标准。

## 在线使用 / Live

开启 GitHub Pages 后，直接打开：

**https://freddiehuhu.github.io/English-Learning-Strategy/**

手机、平板、电脑浏览器都可用，点按钮即可听音频。

## 功能

- **两种模式**：📋 列表 / 🂠 卡片（Anki 式，点击翻面）
- **正面** = 英文单词 + 英文例句；**背面** = 中文释义 + 中文例句
- **四个显示开关**：单词英 / 单词中 / 例句英 / 例句中 —— 可任意隐藏，用于默写自测
- **每卡错误计数** `错 − N ＋`：手动记录听写错误次数，**存在浏览器本地**（换设备不同步）
- **按错误次数排序**（多 → 前），配合「↻ 重排 / 清空计数」
- **四种音频**：单词·英 / 单词·美 / 例句·英 / 例句·美；每个单词读 **3 遍**，开头留白，避免被播放器切掉

## 语音

- 英式：**en-GB-SoniaNeural**；美式：**en-US-AvaNeural**（微软 Azure 神经语音，通过 edge-tts 生成）
- 格式：MP3，24kHz
